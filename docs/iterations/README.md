# 迭代工作记录

每次迭代在此目录下单独建文件夹，存放该迭代的**具体工作内容**（任务清单、验收结果、产物路径、问题记录），与总规划 [`迭代文档.md`](../迭代文档.md) 区分。

## 目录约定

```
docs/iterations/
├── README.md                 # 本说明
├── iteration-0/              # 迭代 0 工作区
│   ├── README.md             # 迭代目标、任务、验收
│   ├── WORKLOG.md            # 执行过程与结论
│   └── artifacts/            # 脚本产出（闭包报告等）
└── iteration-1/            # 后续迭代同理
```
