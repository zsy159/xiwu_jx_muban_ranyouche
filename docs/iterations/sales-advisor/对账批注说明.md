| **浅灰色** | `#D9D9D9` | **金标准直接填数**（无公式） | `金标准直接填数，无公式` | `highlight_commission_summary_deferred_cells` · `static_cells` |
| **浅蓝色** | `#BDD7EE` | **公式含手工**（YAML 登记 + topology 自动检出） | `公式含手工录入，对账暂缓` | `highlight_commission_summary_deferred_cells` · `deferred_cells` |
| **浅橙色** | `#FCE4D6` | **公式形态异常**（如 #REF!）+ 个案说明 | YAML / topology 登记的 `reason` + 可选对账数值 | `add_commission_summary_annotations` |

优先级（后写覆盖填充）：琥珀 mismatch → 灰/蓝 deferred/static → **橙色 anomaly + 批注**。

同一格若同时命中 `static_cells` 与 deferred（含 `wa_parity_deferred`、二网 AH、公式尾项常数），**浅蓝优先**。

> **所有着色单元格均须有 Excel 批注**，悬停即可见含义；琥珀色格另附金标准/系统数值。  
> 已在 deferred 登记的格（如唐操整车绩效、韩柏成保险/加装）标**浅蓝**，不再重复橙色批注。  
> 金标准 D 列含「二网」致 AH 留空、系统仍正确算整车超额时，Hub **权限结余绩效** 标蓝且对账跳过（见 `golden_perf_skips.py`）。  
> 金标准 topology + workbook 检测到的**直接填数格**（无公式，如王熙鸿 H74=1）标**浅灰**。  
> 金标准**公式含尾项常数**（如 `=-140`、`=SUMIFS(...)-100`、`=AH80-100`）由 `collect_topology_manual_formula_cells` 自动标**浅蓝**，不再标橙。  
> 金标准公式含 `#REF!` 等形态异常标**浅橙**并附个案批注（见 `parity_annotation.yaml`）。

## 2. 公式含手工（浅蓝）检出规则

`is_manual_formula_adjustment` / `collect_topology_manual_formula_cells`（`topology_specs.py`）：

| 形态 | 示例 | 说明 |
|------|------|------|
| 纯常数公式 | `=-140`、`=100` | 金标准直接写死数值 |
| 公式 + 尾项加减 | `=SUMIFS(...)+600`、`=SUMIFS(...)-100` | 在标准聚合公式后手工调整 |
| 单元格引用 + 尾项 | `=AH80-100` | 引用上游格后再手工加减 |

**不**视为手工（仍按正常公式对账）：无尾项常数的 `=SUMIFS(...)`、`=W80*0.8` 等。  
含 `#REF!` 的公式走橙色异常批注，不走浅蓝。

检出范围：金标准 提成汇总 数据列（F–P、W–AI 等，同 `HUB_COLUMN_MAP`），与 `collect_topology_static_fill_cells` 相同列集。

合并进 deferred 的来源（`build_reconcile_deferred_cells`）：

1. `wa_parity_deferred`（YAML 登记）
2. 二网致 AH 留空（`erwang_blank_ah_deferred_cells`）
3. topology 公式尾项常数（自动检出）

## 3. 批注内容来源

1. **YAML 登记** — `salary_pipeline/config/parity_annotation.yaml` → `hub_annotations`（橙色个案）
2. **Topology 自动检测** — 金标准 workbook：`collect_topology_manual_formula_cells` → 浅蓝；`#REF!` → 橙色
3. **直接填数检测** — `collect_topology_static_fill_cells`：无公式且有值 → 浅灰
4. **对账数值附注** — 琥珀 mismatch、橙色 anomaly：批注末尾追加 `金标准=…  系统=…  差=…`

代码入口：

- 颜色/批注：`salary_pipeline/utils/excel_format.py`
- 公式检出：`salary_pipeline/calculators/sales_advisor/topology_specs.py`
- 橙色登记：`salary_pipeline/calculators/sales_advisor/parity_annotations.py`
- reconcile 合并：`salary_pipeline/calculators/sales_advisor/registry.py` · `build_reconcile_deferred_cells`
