## Tier B — 可选但建议有（4 张）

不阻塞试算；有对应人员或管理岗行键时建议上传，否则 Hub 管理岗列可能为空或拓扑 `#REF`。

| 工作表 | 标签 | 说明 |
|--------|:----:|------|
| 人员信息 | 可选 | 提供店别/职务/姓名骨架；缺则回退读合并账套内行键 |
| 产品经理提成核算 | 可选 | Hub 拓扑回放 — 销售主管等管理岗整车完成考核（如 AK） |
| 销售管理岗提成依据 新标 | 可选 | Hub 拓扑回放 — 销售管理岗 AK 等列 SUMIF/INDEX 上游 |
| 销售任务及完成率 | 可选 | 公式中间表；系统可读 workbook 结果，亦可由模块重算 |
