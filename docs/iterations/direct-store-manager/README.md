## 算薪公式（门店块）

- 展厅完成率 = min(实际/任务, 110%)
- 展厅提成 = 实际×完成率×单台标准 + 新能源×单台标准
- 渠道完成率同上；渠道提成 = 完成率×单台标准×实际 + 新能源任务×单台标准
- **提成合计** = 台次提成合计 + 附加收入提成 + 固定绩效 + 指定车型考核提成  
- 附加收入来自子表下行块（P=V{row+10}），当前从 Excel 预填
