**日期：** 2026-06-25

## 结论

| 项目 | 归属层 | 何时做 |
|------|--------|--------|
| `整车完成考核`（AK） | Hub / `direct_store_manager_performance` | ✅ 已完成 |

## 依据

发薪表公式（朱剑波 / `直营店提成-发` 行 10）：

```
AG = SUMIF(银河B直营店提成!AG) + SUMIF(银河A提成!AH)
AH = AE - AF + AG
实际发放 = AH - AP - AR
```

- **不在 `提成汇总`**：`代发放绩效` 从未写入 Hub W–AI 列。
- **引擎可复用**：`XwPayoutFormulaEngine` 已在 `XW提成-发` 实现银河双表 SUMIF；直营店渠道只需换表 + `直营店基本`。
- **避免重复**：不应把银河逻辑塞进 `direct_store_manager_performance`。
